# SSU LMS 확장프로그램 설치 가이드

## 설치 방법

### 1단계 — 개발자 모드 활성화

1. Chrome 주소창에 입력:
   ```
   chrome://extensions
   ```
2. 우측 상단 **"개발자 모드"** 토글을 **ON**으로 전환

### 2단계 — 확장프로그램 로드

3. 좌측 상단 **"압축해제된 확장 프로그램 로드"** 버튼 클릭
4. 이 폴더(`ssu-lms-bypass`)를 선택 → **확인**

### 3단계 — 설치 확인

5. 확장프로그램 목록에 **"SSU LMS"** 가 표시되면 완료
